## Setup

Just follow the steps below carefully and your server will be fully protected in no time:

<Steps>
  <Step title="Invite Zynrax">
    [Click here](https://discord.com/oauth2/authorize?client_id=1402377137087840378&permissions=8&response_type=code&redirect_uri=https%3A%2F%2Fdiscord.gg%2FcGNxXQNUWg&integration_type=0&scope=bot+guilds) to invite Zynrax into your server.
  </Step>
  <Step title="Setup Zynrax's role position">
    Go to Server Settings > Roles > Drag Zynrax's default role to the top of the roles list.
  </Step>
  <Step title="Initialize Setup">
    Run `?security` command in any channel.
  </Step>
</Steps>

## Role Positioning

<Tip>
  You need to move Zynrax's role and place it at the top in roles list, Make sure it is Zynrax's default role, not any other role given to Zynrax.
</Tip>

## Permission

<Note type="info" icon="circle-info">
  Make sure you invite Zynrax with the proper permission it asked for `ADMINISTRATOR`
</Note>

<Warning>
  Only the **Server Owner & Extraowner** is able to setup Zynrax using the `?security` command.
</Warning>


That's it !!
